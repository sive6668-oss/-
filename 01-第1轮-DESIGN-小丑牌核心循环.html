3.6

按钮规范（v4 最终版）

中文友好 + 多层渐变质感 + hover/active 反馈

所有按钮统一基础样式 .px-btn，按职责分配颜色。

基础样式

.px-btn {
  min-height: 52px;
  padding: 14px 26px;
  border-radius: 12px;
  font-family: 'Inter', 'PingFang SC', 'Microsoft YaHei', sans-serif;
  font-size: 16px;
  font-weight: 800;
  letter-spacing: 1.5px;
  text-shadow: 0 1px 2px rgba(0,0,0,.45);
  border: 2px solid rgba(0,0,0,.35);
  transition: all 0.15s ease;
  cursor: pointer;
}
.px-btn:hover:not([disabled])  { transform: translateY(-2px); filter: brightness(1.1) saturate(1.15); }
.px-btn:active:not([disabled]) { transform: translateY(2px);  filter: brightness(0.92); }
.px-btn[disabled]              { opacity: 0.45; cursor: not-allowed; filter: grayscale(0.3); }

按职责分配颜色

按钮背景渐变（180deg, 顶→中→底）hard shadow
出牌（主，绿）#34d399 → #10b981 → #0596690 5px 0 #047857
弃牌 / 重开（红）#fb7185 → #ef4444 → #dc26260 5px 0 #991b1b
排序（ghost 紫蓝）rgba(99,102,241,.28) → rgba(67,56,202,.20)0 3px 0 rgba(67,56,202,.4)
设置（橙）#fbbf24 → #f59e0b → #d977060 4px 0 #92400e
商店跳过（蓝紫）#818cf8 → #6366f1 → #4338ca0 5px 0 #312e81

每个按钮额外加 inset 顶部高光 inset 0 1px 0 rgba(255,255,255,.3)，让按钮看起来有立体浮雕感。

⛔️ 禁忌
  • 不要用 Press Start 2P 渲染按钮中文（v3 用 9px 像素字几乎看不清，v4 已切到 Inter 16px）
  • 不要把按钮高度做扁（< 44px）—— 移动端触摸不友好
  • 不要省略 hover/active 反馈 —— 学员第一感觉就是"按钮没反应"
