一、产品概述

Balatro 风格的「扑克 + 小丑牌组合」游戏的核心循环版本。玩家从 0 金币、0 张小丑牌开始，通过出 5 张以内扑克牌型 + 持有小丑牌触发的加成攒分，4 手内打到当前盲注的目标分则通关，进入商店购买新小丑牌增强能力，依次推进 3 个递增盲注关卡。3 关全过通关，任意一关 4 手内打不到则失败。

本轮的目标是「跑通完整玩法循环」，让玩家亲眼看到「选牌 → 出牌 → 触发 Joker → 计分 → 通关 → 进商店 → 买卡变强 → 打下一关」这条主链路。

二、核心玩法循环

开局 $0 / 0 Joker → 小盲注 300 分 → 通关拿钱 → 商店买 / 跳过 → 中盲注 500 → 商店 → 大盲注 800 → 🎉 通关全部

任意一关 4 手内未达目标分 → 💀 失败。

1.2 计分公式 锁定

得分 = (牌型基础筹码 + 所有出牌的点数之和) × 牌型基础倍率
       → 然后按持有 Joker 数组的顺序，每张 Joker 的 effect 依次修改 (chips, mult)
       → 最终得分 = 最终 chips × 最终 mult

点数计算规则：A = 11、J/Q/K = 10、其他 = 数字本身。

五、关键交互与动效

5.1 出牌计分动画时间线 v7 本轮全做（含飞牌 + 飞字）

🎬 v6 → v7 调整：4 类动效全部前置到本轮 v5 把飞牌 / 飞字 / 计分粒子留到第 3 轮，v6 跟着砍。学员反馈「太 low」。v7 把以下 4 类动效全部纳入本轮必做：① 选中牌飞向出牌区；② 出牌时逐张高亮 + 蓝色 chips 飞字；③ Joker 触发金光 + 红色 mult 飞字；④ 发牌从牌堆飞入手牌。第 3 轮只剩配乐 + 粒子升级（通关撒金币、爆分粒子等更高级的特效）。

出牌不是「按一下分数瞬间变」，而是按以下步骤逐步推进，让玩家看清每一步：

  1. 必做　选中的牌（1-5 张）飞向出牌区中央位置（≈350ms）—— 用 Vue Transition JS hooks + GSAP/setTimeout 实现，clone 元素 + fixed 定位 + transform 飞过去
  2. 必做　显示牌型名 + sidebar 锁定初始 chips / mult（牌型基础值）
  3. 必做　从左到右逐张高亮（每张 150ms，上移 18px + 蓝光），sidebar chips 跳动累加；每张牌上方飞出蓝色 +N 字向 chips 块（飞行 400ms，淡出消失）
  4. 必做　按 ownedJokers 顺序，被触发的 Joker 上移 + 金光（800ms）；每个 Joker 上方飞出红色 +N Mult 或 ×N 字向 mult 块（飞行 400ms，淡出消失）
  5. 必做　HAND 块整体闪一下，中央弹出大金字 chips × mult = score（停留 800ms）
  6. 必做　blindScore 累加（用 RAF 数字插值跳动 ≈ 600ms），handsLeft - 1
  7. 必做　淡出出牌区，新牌从牌堆位置飞入手牌槽位（每张错峰 60ms，每张飞行 400ms）

时间总轴：1 飞牌 350ms → 2 设牌型 ≤ 200ms → 3 逐张高亮 750ms（5 张 × 150ms）→ 4 Joker 触发 300ms × N → 5 公式爆出 800ms → 6 blindScore 累加 600ms（与 5 重叠）→ 7 补牌发牌 400ms → 共约 2.6-3.2 秒一手（取决于 Joker 数量）。

5.2 发牌动画 v7 本轮必做

所有新牌（初始 8 张 / 出牌后补 / 弃牌后补 / 新一关初始）必须从牌堆当前位置（PlayArea 右下角）飞入手牌目标位置（不能原地淡入），每张错峰 60ms，飞行 400ms，用 Vue JS hooks + getBoundingClientRect() 实时算偏移。

5.3 Joker 触发反馈

被触发的 Joker 卡必须有视觉反馈：translateY(-18px) + scale(1.15) + 16px 金光 box-shadow + 持续约 800ms 后恢复（不要瞬切回去，加 ease-out 过渡）。

八、验收清单

初始状态

  • 页面打开即可玩，不需要登录
  • HUD 显示：「第 1/3 关 · 🔵 小盲注 · 目标 300 · 当前 0 · 4 手 · 3 弃 · $0」
  • Joker 槽 5 个全空（不是预置 Joker）
  • 8 张手牌从右下角牌堆飞入
  • 出牌按钮在未选时灰色禁用，弃牌按钮在未选时灰色禁用

出牌循环

  • 选 1-5 张点出牌：被选牌飞到 Played hand 区，按时间线逐张高亮 + 浮字累加 + 最后爆出总分
  • 分数累加到 Round Score，剩余手数 -1，新牌从右下角飞入补满 8 张
  • 累计 ≥ 300 分 → 自动进商店，金币变成 $5-$9（取决于剩多少手）
  • 4 手用完没达标 → 「💀 失败」+ 显示卡在第 1 关

商店循环

  • 商店显示 3 张随机 Joker，按钮文案 / 灰显逻辑符合 3.3
  • 买完一张 → Joker 进槽位，金币扣减，按钮变「已售出」
  • 「跳过 →」按钮永远可点，点完进下一关
  • 第 2 关目标 500、第 3 关目标 800
  • 3 关全过 → 「🎉 通关全部」+ 显示金币 + 持有 Joker

Joker 触发

  • 持有「梅花爱好者」时打含 ♣ 的牌 → 卡片金光 + 倍率 ×4
  • 持有「皇家头牌」时打含 J/Q/K → 倍率 ×10
  • 持有「同花顺大师」时打同花顺 → 倍率 +50
  • 多张 Joker 同时符合条件 → 按持有顺序依次叠加生效

弃牌

  • 选 1-5 张 + 还有弃牌次数 → 「弃牌 (X)」按钮可点
  • 弃牌后：被选牌移除 → 从右下角牌堆补新牌到 8 张
  • handsLeft 不变，discardsLeft -1，Round Score 不变
